# Edge-Traffic-Density-Control

# 🚦 Edge-Based Smart Traffic Density & Signal Controller

## 📌 Project Overview
This project implements an intelligent traffic signal control system using edge AI on Jetson Nano. The system analyzes real-time traffic video to estimate vehicle density and dynamically adjusts traffic signal timing to reduce congestion.

---

## 🎯 Objective
- Detect vehicles using computer vision
- Estimate traffic density in real-time
- Dynamically control traffic signals
- Reduce waiting time and congestion
